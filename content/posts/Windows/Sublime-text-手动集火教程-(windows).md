1. 备份原始 sublime_text.exe 到 sublime_text.exe.bak 
2. 转到 https://hexed.it/ -> 单击“打开文件”(sublime_text.exe)
3. 查找十六进制：80 78 05 00 0f 94 c1
4. 替换为 c6 40 05 01 48 85 c9
5. 另存为（sublime_text.exe）替换源文件, 完成

    Happy Code (^-^)
